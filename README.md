# MusicCloud
Muisc Cloud is project, that allow user to add/update/delete albums and its songs. Implemented using Django(Python)

## Getting Started
**Usage**
```
#  cd MusicCloud-master/
# python manage.py runserver
```
 Open web browser
  ```
127.0.0.1:8000/music/
  ```
